I’ve received many queries for this question. I am going to answer it now.

To start with programming:→

1)Pick up a language,say, C++

2)Get introduced to the language and learn its basic syntax . Watch video tutorials of :

a) C++ Programming Tutorials (HINDI/URDU) - YouTube

b)C++ by Saurabh Shukla - YouTube

to get the above thing clear.

3)Focus heavily on the logic of:-

1)If-else statement.

2)Loop

3)Array

Solve and learn these sections from here:-

Decision Making Statement

4)Now, if you are enrolled in a computer-science course, you will find basic good problems based on:-

a)Checking if a number is palindrome/prime number,etc.

b)Making different types of patterns

c)Find sum of different type of series using Array and Loops! :O

Solve them as much as you can from : - Sitesbay - Bay of Websites

Now.you are done with the programming part which is needed to start competitive programming.

Now, the real game starts.

For competitive programming :

  1. Go to Array Data Structure - GeeksforGeeks and solve as many problems as possible on array.
  2. Go to String Data Structure - GeeksforGeeks and solve as many problems as possible on string
  3. Also solve problems on Hackerearth :- Basics of Input/Output Tutorials & Notes | Basic Programming | HackerEarth and Hackerrank:-Solve Algorithms Code Challenges
  4. Repeat the above process for 2–3 months.
  5. Start participating in the contests of HackerRank , Programming Challenges & Coding Competitions on HackerEarth , Programming Competition,Programming Contest,Online Computer Programming and Codeforces-Div-2
  6. Most important:-Regularly up-solve the problem(s) which you couldn’t solve during the contest. You will always learn the new trick(s)/algorithm(s) needed,very fast,this way!
  7. Now, is the time to learn data structures,algorithms , dynamic programming and some Math ! :D
  8. Learn it from Project Euler,Geeksforgeeks and Hackerearth.

Advanced level:

1)Solve Russian Coding problems here :- Timus Online Judge

2)Practice DIV-1 problems of codeforces and topcoder regularly.

Most of the above, worked out very well for me, see if it helps you out too! :)

And wait,also in case if you are not able to solve a problem, then send me its link, my team will prepare a fresh editorial for it :)

  • Pick a language of your choice….. (I advise not to pick Python as it provides so many shortcuts that, as a beginner, your concepts will remain unclear !). Preferably, C++ or Java !
  • Understand core concepts. (This is a must, even for an experienced Programmer).
  • Start practising with basic codes so that you can adapt to programming and get comfortable with the syntax.
  • Once your basics are clear, you can start with Hackerrank (There are n number of coding platforms but I believe that for someone relatively new to online coding, Hackerrank is the right one).
  • Move from basics to average.
  • Then Learn advanced concepts like Dynamic Programming, Data Structures etc.
  • Practice, Practice, Practice !!!!!
